# Kigali Guide 🌍

## Project Description
A multi-page website that lists things to do and places to eat around 
Kigali, Rwanda. The website is informative, well-structured, and styled 
using CSS.

## HTML Pages Included
- **index.html** - Home page linking to all destination pages
- **destination1.html** - Nyamirambo neighborhood guide
- **destination2.html** - Gisozi and Kigali Genocide Memorial
- **destination3.html** - Kigali City Tour with video, audio and map
- **destination4.html** - Best restaurants and food in Kigali

## What This Project Demonstrates
- Correct HTML page structure and semantic elements
- Multi-page navigation using internal and external links
- Tables, ordered and unordered lists
- Image embedding with alt text
- Video, audio and embedded Google Map
- Separate CSS file for each HTML page
- GitHub version control with meaningful commits
